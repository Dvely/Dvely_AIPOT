
  # AIPOT Frontend v1.2

  This is the frontend code bundle for AIPOT v1.2. The original design file is available at https://www.figma.com/design/Pqc0NrzK7RZolT0Cqpftam.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create a `.env` file when you want to change API target:

  ```bash
  VITE_API_BASE_URL=http://localhost:3000
  ```

  If omitted, frontend uses `http://localhost:3000` by default.

  Run `npm run dev` to start the development server.
